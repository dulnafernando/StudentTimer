# Study Break Timer App

## Project Description
The Study Break Timer App is an Android application designed to help users manage study intervals and breaks using the Pomodoro technique. The application helps students stay focused by alternating between study sessions and short breaks while keeping track of completed sessions.

---

## Features
- **Secure Login and Registration**
  - User authentication system
  - Passwords stored securely using hashed encryption

- **Real-time Study / Break Timer**
  - Timer for study sessions
  - Automatic break intervals

- **Session History**
  - Records all study sessions
  - Supports CRUD operations (Create, Read, Update, Delete)

- **User-Specific Data**
  - Users can only see and manage their own session records

---

## Technologies Used
- Android SDK (Java / Kotlin)
- SQLite Database
- Material Design UI
